🌐

Real-time Translation

Translate course pages with packaged dictionaries, local cache, and Google Translate. No account or human-check is required.

🧰

Local Learning Tools

Use flashcards, bookmarks, recent lessons, progress, outline, reading aids, and PDF export with state stored in your browser.

🎬

Auto-Subtitles

Course videos automatically activate translated subtitles when you play them. No manual toggle needed.

🛡️

Protected Terms

1,100+ curated terms per language. Brand names and AI terminology stay accurate — no more "Prompt" → "신속한" errors.

🌙

Dark Mode

Full dark theme for the course header, lesson content, and SkillBridge panels. Toggle with one click.

💚

100% Free

The CWS edition needs no account, email, password, user API key, or human-check.

🔒

Privacy-First

No SkillBridge accounts or analytics. The unpublished candidate runtime makes no Puter, Gemini, Claude-model, or AI Tutor request.

🎓

Exam Mode

Auto-detects certification exams. Quiz answer choices stay untranslated, and recognized proctored routes disable the extension.

⌨️

Keyboard Shortcuts

Ctrl+Shift+S (sidebar), Ctrl+Shift+F (flashcards), Ctrl+Shift+L (dark mode), Ctrl+Shift+/ (help), Escape (close).

🦊

Cross-Browser

Works on Chrome, Edge, Brave, Arc, and other Chromium browsers. Firefox support is in beta — run npm run build:firefox to try it.
